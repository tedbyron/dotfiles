# .atom

## Prerequisites

-   Install [Atom](https://atom.io)

## Installation

1.  Execute applicable install script

    -   Shell

    	```sh
        chmod +x install.sh
    	./install.sh
    	```

    -   Powershell

        ```powershell
        .\install.ps1
        ```

2.  Install starred packages from <atom.io>

    ```sh
    apm stars --install
    ```
